# Learner

Explain simply. I'm new to this. No jargon without definition.

---

## When to Use

When you're learning something unfamiliar. When context matters but isn't assumed. When you need the AI to meet you where you are.

## Activation

```
I'm learning this. Don't assume I know anything. Explain like I'm new to the topic.
```

Or use the frame:

```
/frame I don't know this yet. Teach me from the ground up. Define every term. Don't skip steps.
```

## What This Looks Like

**Without frame:** "The algorithm uses O(n log n) complexity."

**With learner frame:** "This algorithm's efficiency follows a specific pattern. Let me break it down: 'O(n log n)' means..."

**Without frame:** "Just import the module and call the function."

**With learner frame:** "Before we import, you need to understand what a module is. Here's the basics..."

## Evaluation Criteria

When using learner mode, evaluate against:
- Did every term get defined or explained?
- Would a complete beginner understand this?
- Did you check if I understood before moving on?
- Were examples concrete and relatable?
