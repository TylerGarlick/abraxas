# Evaluate: Research

Verify: source credibility, date freshness, methodology soundness, citation accuracy.

---

## When to Use

When doing research. When you need to trust your sources. When accuracy matters more than speed.

## Activation

```
I'm researching this topic. Before I use any information, verify: are sources credible? Is the data current? Are conclusions sound?
```

Or use the frame:

```
/frame Treat this as research. Verify every claim's source. Flag anything outdated. Question methodology. Check if citations actually support what's claimed.
```

## Evaluation Criteria

When evaluating research, assess against:

| Dimension | What to Look For |
|-----------|------------------|
| Source Credibility | Who published? What's their expertise? Any known biases? |
| Date Freshness | When published? Has the field changed since? |
| Methodology | How was this determined? Is the approach sound? |
| Citation Accuracy | Do claims match sources? Any misrepresentation? |
| Replicability | Could this be verified independently? |

## What This Looks Like

**Without criteria:** "According to a study..."

**With evaluate-research:** "This study is from 2015 — significant developments have happened since. Also, the methodology has been criticized by more recent work. Here's the current state of the field..."

**Without criteria:** "Research shows..."

**With evaluate-research:** "The source you cited actually says the opposite of what you're claiming. Let me show you the actual finding..."

## Output Format

Structure your evaluation as:

```
## Source Credibility
- Source: [name/link]
- Credibility: [high/medium/low] — [reasoning]
- Expertise: [relevant background]

## Currency
- Published: [date]
- Status: [current/outdated/field has evolved]

## Methodology Assessment
- Approach: [what they did]
- Soundness: [strengths/weaknesses]
- Limitations: [what's not addressed]

## Citation Accuracy
- [If claims match sources or misrepresent]

## Reliability Verdict
[High confidence / Medium confidence / Low confidence — explain why]
```

## Success Criteria

- Did you verify, not just cite?
- Did you check dates and currency?
- Did you assess methodology soundness?
- Did you catch any citation misrepresentations?
- Did you give a clear confidence level?
