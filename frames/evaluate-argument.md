# Evaluate: Argument

Assess: logical soundness, evidence quality, hidden assumptions, alternatives ignored. Find weaknesses first.

---

## When to Use

When evaluating claims. When you need to test reasoning, not just accept it. When you want to find flaws before committing to a position.

## Activation

```
I want you to evaluate my argument for weaknesses. Find the holes. Don't help me strengthen it yet — first, show me what's broken.
```

Or use the frame:

```
/frame This is a critique session. Assess my argument for logical fallacies, weak evidence, unstated assumptions, and alternatives I haven't considered.
```

## Evaluation Criteria

When evaluating arguments, assess against:

| Category | What to Look For |
|----------|------------------|
| Logic | Circular reasoning, false dichotomies, strawman, ad hominem |
| Evidence | Source credibility, cherry-picking, correlation vs causation |
| Assumptions | Unstated premises, hidden constraints, unexamined biases |
| Alternatives | Options ignored, other perspectives dismissed, tradeoffs unexplored |

## What This Looks Like

**Without criteria:** "That's an interesting point."

**With evaluate-argument:** "Your conclusion assumes X is true, but you haven't established X — it's just implied. Also, you dismissed option Y without addressing why it fails. Here's the gap in your reasoning..."

**Without criteria:** "I see your point."

**With evaluate-argument:** "You're conflating correlation with causation here. The data shows A and B together, but you haven't shown A causes B."

## Output Format

Structure your evaluation as:

```
## Logical Issues
- [Specific flaw with location in argument]

## Evidentiary Issues
- [Weakness in evidence or source]

## Hidden Assumptions
- [What must be true for this to work]

## Alternatives Ignored
- [Other positions not addressed]

## Verdict
[Does the argument hold? What's missing to make it strong?]
```

## Success Criteria

- Did you find actual weaknesses, not just surface-level nits?
- Did you identify what's needed to strengthen the argument?
- Did you distinguish between fatal flaws and fixable gaps?
- Did you consider alternatives the argument ignores?
