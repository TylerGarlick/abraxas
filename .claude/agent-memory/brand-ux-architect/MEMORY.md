## Brand Identity

### Name Etymology
- *Abraxas* = Gnostic cosmological symbol; totality of all forces; 365 in Basilidean gematria
- Chosen to represent an AI orchestration layer that unifies heterogeneous capabilities
- Carries both solar and chthonic symbolism — integration of dualities

### Aesthetic Registers
1. **Symbolic**: Mythology, cosmology, alchemy, Jungian psychology, phenomenology — earned metaphor
2. **Rigorous**: Technically precise, epistemically disciplined, calibrated claims
3. **Minimal**: Spare, structured, no promotional energy

### Voice Principles
- Precise, not verbose — every sentence earns its place
- Non-promotional — states what things are, doesn't sell them
- Conceptually confident — introduces complex ideas without apology
- No superlatives: never "powerful", "amazing", "cutting-edge"

## Five-Skill Coherence Model (2026-03-05)

**Epistemic Pipeline:**
1. **Agon** — Opposition generates truth through ritualized contest (two positions, asymmetric priors, convergence analysis)
2. **Janus** — Opposition distinguished and labeled at threshold (Sol/Nox faces, `[KNOWN]`/`[INFERRED]`/`[UNCERTAIN]`/`[UNKNOWN]`)
3. **Honest** — Labels stated plainly in everyday language (no mythological framing, accessible anti-hallucination)
4. **Aletheia** — Labels validated through retrospective disclosure (epistemic ledger, calibration over time)
5. **Abraxas Oneironautics** — All material integrated into lived practice (35-command alchemical system)

**Naming coherence:**
- Greek philosophy dominates (Agon, Aletheia, Oneironautics)
- Latin tradition (Janus)
- Plain English virtue (Honest)
- This diversity is intentional — reflects varied epistemic registers

**Obscurity gradient (correct):**
- Honest = immediately accessible
- Janus = recognizable, slight learning curve
- Agon = requires explanation, learnable
- Aletheia = requires serious explanation, highly rewarding
- Abraxas Oneironautics = requires full philosophical context, maximum symbolic weight
