## About

**Abraxas** is a small CLI application offering:

- a `Click`-based CLI (`abraxas`) to run commands like `info`, `db-test`, and `serve`
- an ArangoDB client wrapper using the `python-arango` driver
- a lightweight MCP server implementation using `asyncio` and `pydantic`

For more details, pick a section above to get started. If you need the full project readme, see the [root README](../README.md).
